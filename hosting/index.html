Hosting

So you've created a website. Great! Now what? This page goes over two types of hosts - SiteGround and GitHub. SiteGround (a web host) can make your site live, while GitHub can host your code to make project collaboration easier.

SiteGround

Where this website is

SiteGround offers a range of "hosting" services to its users - among them web hosting. This allows a client to upload a site for others to view; an essential function for web development. This website (Legal Development) is hosted on SiteGround's servers.

Unlike the other technologies this site focuses on, SiteGround is not open source; to use SiteGround's services, one must first purchase them.

GitHub

The elephant of stuff part 2

GitHub combines the "Git" technology with collaborative qualities and a clean UI. As stated on its "GitHub Guides" section, it allows people of all coding backgrounds to "work together on projects from anywhere" - a helpful quality for managing programs.

On the "Terms" section of its website, GitHub states that it "claims no intellectual property rights" over whatever a user puts on its servers.
