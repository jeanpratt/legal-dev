Why it's important

Bug fixing

“As a large number of users can access and change the code, bugs tend to be more visible and more rapidly corrected.”

James A. J. Wilson; Benefits of open source code
Customization

“[…] Open source software can be readily adapted to meet specific user needs.”

James A. J. Wilson; Benefits of open source code
Translation

“With access to the source code it is easy to translate the language of the software interface.”

James A. J. Wilson; Benefits of open source code
Learning

“[…] Open source code provides an excellent resource from which to learn, and open source projects provide a practical environment in which to test [one’s] skills.”

James A. J. Wilson; Benefits of open source code
Cost

“Many open source programs can be obtained at no cost or at a very low cost.”

James A. J. Wilson; Benefits of open source code
Competition

“The introduction of competition into the market for customizations […] forces suppliers to offer high quality at a competitive price.”

James A. J. Wilson; Benefits of open source code
